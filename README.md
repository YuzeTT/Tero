## 介绍

Tero是一个解决问作业困难和被问作业烦人的Web项目。

管理员发布作业后，同学绑定班级后可以查看今日作业。

## 技术栈

前端
  - Vue3
  - ElementUI

后端
  - Python Flask

## 开发

use yarn
```
yarn
yarn serve
```

use npm
```
npm install
npm serve
```

## 构建

use yarn
```
yarn build
```

use npm
```
npm build
```
